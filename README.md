# 🚦 AI Smart Traffic Control System

An AI-powered traffic congestion monitoring and smart signal control system built using Python, Tkinter, OpenCV, and Machine Learning techniques.

---

## 📌 Features

- 🚗 Vehicle density detection
- 📊 Real-time traffic analysis
- 🧠 AI-based signal timing adjustment
- 📷 Camera feed simulation
- 📈 Traffic statistics dashboard
- 🖥️ User-friendly GUI

---

## 🛠️ Technologies Used

- Python
- Tkinter
- OpenCV
- NumPy
- Pandas
- Machine Learning

---

## ⚙️ Installation

```bash
git clone https://github.com/YOUR_USERNAME/ai-smart-traffic-control-system.git

cd ai-smart-traffic-control-system

pip install -r requirements.txt

python main.py
```
